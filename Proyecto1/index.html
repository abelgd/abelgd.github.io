Lenovo Ideapad Gaming 3 15ACH6 (82K2)

Autor: Abel García Domínguez

La BIOS (Basic Input/Output System) y la UEFI (Unified Extensible Firmware Interface) son los sistemas de firmware que inicializan el hardware del equipo y gestionan el arranque del sistema operativo. La UEFI es la evolución de la BIOS tradicional y ofrece un entorno más seguro, compatible con discos de gran capacidad y con funciones como el arranque seguro (Secure Boot).

Bastionar la BIOS/UEFI consiste en configurar correctamente sus parámetros de seguridad para proteger el equipo frente a accesos no autorizados o manipulaciones durante el arranque.

De esta forma, se refuerza la integridad del sistema y se reduce el riesgo de ataques que puedan comprometer el ordenador antes de iniciar el sistema operativo.

En este menú se configuran todas las contraseñas y permisos de seguridad de la BIOS/UEFI.

Pasos:

  1. Contraseña de administración de la BIOS
    • Accede al menú Security → Admin Password.
    • Selecciona "Set Administrator Password" y crea una contraseña segura.
    • Guarda los cambios y confirma.
  2. Contraseña de usuario de la BIOS
    • Dentro del mismo menú Security → User Password.
    • Configura una contraseña diferente a la de administrador.
    • Introduce la nueva contraseña y confírmala cuando el sistema lo solicite.
    • Esta contraseña será requerida para entrar al BIOS sin privilegios de administrador.
  3. Contraseña de arranque del dispositivo (Power-On / HDD)
    • Todavía en Security, localiza Set Hard Disk Passwords.
    • Introduce y confirma la contraseña de arranque cuando el sistema lo indique.
    • Esta contraseña se solicita al encender el equipo y protege el arranque del disco.

Nota:

Tanto en el paso 2 como en el paso 3, no se incluyen capturas del proceso de introducción y confirmación de contraseñas, ya que el simulador de la BIOS de Lenovo utilizado para la elaboración de esta guía no permite acceder a la interfaz donde se establecen las contraseñas de usuario o de arranque.
En un entorno físico real, al seleccionar estas opciones, el sistema solicitará introducir y confirmar las contraseñas de manera secuencial antes de guardarlas.

Este menú controla desde qué dispositivos puede arrancar el ordenador y en qué orden, así como la seguridad del arranque.

Pasos:

  1. Boot Mode
    • Selecciona UEFI para garantizar compatibilidad con Secure Boot, discos GPT y arranque seguro moderno.
    1.1 Orden de arranque (Boot Priority Order)
    • En este modelo, la BIOS no muestra una lista manual de prioridad de arranque, ya que al usar modo UEFI, la prioridad la gestiona automáticamente el sistema operativo mediante Windows Boot Manager.
    • Asegúrate de que el equipo está configurado en modo UEFI (paso anterior) y que Windows Boot Manager aparece como opción de arranque en la pestaña "EFI".
    • De esta forma, el sistema siempre arrancará primero desde el disco interno autorizado.
  2. Fast Boot
    • Puedes activar Fast Boot para un arranque más rápido.
    • Nota: desactivarlo permite acceso a teclas de BIOS y diagnóstico; activar o desactivar según tu política de seguridad.
  3. USB Boot
    • Deshabilita el arranque desde USB si no necesitas iniciar desde medios externos.
    • Recomendación: mantenerlo desactivado para evitar arranques no autorizados.
  4. PXE Boot to LAN
    • Deshabilita el arranque desde la red si no se requiere.
    • Esto previene que el equipo arranque con sistemas operativos no autorizados desde la red.
  5. IPV4 PXE First
    • Desactívalo, ya que no se utilizará arranque por red en entornos domésticos o personales.
  6. EFI
    • Confirma que el arranque se realiza en modo EFI para Secure Boot y discos GPT.
  7. Secure Boot
    • Localiza la opción Secure Boot (pestaña Security).
    • Asegúrate de que esté configurada como Enabled.
    • Esto evita que el sistema arranque con sistemas operativos o controladores no firmados digitalmente.

Nota de seguridad: Al desactivar Fast Boot, USB Boot o PXE Boot, algunas opciones relacionadas pueden desaparecer del menú. Esto es normal y forma parte de la configuración más segura del arranque.

En este menú se confirman y aplican todos los cambios realizados en la BIOS/UEFI.

Pasos:

  1. Exit Saving Changes
    • Selecciona esta opción para guardar todas las configuraciones aplicadas en los menús Security y Boot y salir de la BIOS.
    • Confirma cuando se te pregunte si deseas guardar los cambios.
  2. Save Changes
    • Alternativa a la opción anterior si solo quieres guardar los cambios sin salir inmediatamente de la BIOS.
  3. Comprobación del arranque
    • Una vez reiniciado el sistema, verifica que:
      • Se solicita la contraseña de arranque si configuraste una.
      • Solo el disco interno aparece como dispositivo de arranque.
      • Secure Boot está activo.

Resultado esperado:
 El sistema arranca de forma segura, solicitando las contraseñas configuradas y sin permitir el arranque desde dispositivos no autorizados. Por lo tanto, el equipo queda protegido frente a accesos no autorizados al firmware, garantizando un arranque seguro y controlado por el usuario.
